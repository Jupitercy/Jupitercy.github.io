chcon can't apply partial context to unlabeled file varwwwtemp报错

发表于2022-12-16|更新于2023-01-31|技术Linux常见问题
|字数总计:127|阅读时长:1分钟|阅读量:|评论数:

命令：chcon -R -t httpd_sys_content_t /var/www/temp
报错内容：chcon: can't apply partial context to unlabeled file /var/www/temp

解决方法：修改/etc/selinux/config

修改内容SELINUX=disabled为SELINUX=enforcing

重启服务器reboot，然后重新启动httpd服务，再次执行命令就OK了。
